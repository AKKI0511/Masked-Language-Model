# Self-Attention Visualization using BERT

This repository contains a Python script for visualizing self-attention scores using a pre-trained BERT (Bidirectional Encoder Representations from Transformers) model. Self-attention is a mechanism in transformers that allows the model to weigh the importance of different words in a sentence when making predictions.

## Requirements

To run the script, you need to have the following installed:

- Python 3
- TensorFlow
- Hugging Face Transformers library
- PIL (Python Imaging Library)

You can install the required packages using pip:

```bash
pip install tensorflow transformers Pillow
```

## Usage

1. Clone the repository:

```bash
git clone https://github.com/AKKI0511/Masked-Language-Model.git
cd Masked-Language-Model
```

2. Run the script:

```bash
python mask.py
```

You will be prompted to enter a text containing a mask token (`[MASK]`). The script will then generate predictions for the masked token and visualize the self-attention scores for each layer and attention head in the BERT model.

## Configuration

- `MODEL`: Specify the BERT model to use. The default is `"bert-base-uncased"`.
- `K`: Number of predictions to generate for the masked token.
- `FONT`: Path to the font file for drawing text on attention diagrams.
- `GRID_SIZE`: Size of each grid cell in the attention diagrams.
- `PIXELS_PER_WORD`: Size of each word in pixels in the attention diagrams.

You can modify these constants in the script according to your requirements.

## Visualizing Attention

The script generates attention diagrams for each attention layer and head in the BERT model. Each diagram shows the self-attention scores for a single attention head. The cells in the diagram are shaded based on the attention weights, with lighter cells corresponding to higher attention scores.

The diagrams are saved as PNG files with filenames that include the layer number and head number.
